本文最后更新于：2022年1月3日 上午

Selenium 处理弹窗以及iframe 处理

1、弹窗

在使用selenium的过程中，可能会遇见弹窗，常见的弹窗类型有：alert 、confirm 或prompt 弹窗。selenium 为我们提供了相应的方法：

  • getText()：返回alert/confirm/prompt 中的文字信息。
  • accept()： 接受现有警告框。 点击【确认】
  • dismiss()：解散现有警告框。点击【取消】
  • sendKeys(keysToSend)：发送文本至警告框。
  • keysToSend：将文本发送至警告框。

通过driver.swtichTo().alert();切换到alert 

public static void main(String[] args) {
       System.setProperty("webdriver.chrome.driver", "C:\\Program Files (x86)\\Google\\Chrome\\Application\\chromedriver.exe");
       ChromeOptions options = new ChromeOptions();
       // 接管本地浏览器
       options.setExperimentalOption("debuggerAddress", "127.0.0.1:9333");
       WebDriver driver = new ChromeDriver(options);
       WebDriverWait w = new WebDriverWait(driver, 10);
       try {
           Thread.sleep(1000);
       } catch (InterruptedException e) {
           e.printStackTrace();
       }
       try {
           //alert
            driver.switchTo().alert();
            System.out.println(driver.switchTo().alert().getText());
            driver.accept();
           //confirm
            driver.switchTo().alert().dismiss(); 
            driver.switchTo().alert().accept();
           //promot
           driver.switchTo().alert().sendKeys("hello selenium");
           driver.switchTo().alert().accept();
       } catch (Exception e) {
           System.out.println(e.getMessage());
       }

   }

2、iframe

通过driver.switchTo().frame()切换到iframe，具体通过 id、clssname等，实在不行就使用xpath(),巨好用，

driver.switchTo().frame(int index);//如果有多个iframe，可获取对应顺序
driver.switchTo().frame(String nameOrId);
driver.switchTo().frame(webElment e);

切到frame中之后，我们便不能继续操作主文档的元素，这时如果想操作主文档内容，则需切回主文档。

driver.switchTo().defaultContent();

根据我以往的经验来看，在出现或者可能会出现iframe或者alert的地方，要设置等待。
